# Automated Compiler-in-the-Loop Prompt Optimization for ScaFi

This repository implements an automated, compiler-guided prompt optimization pipeline for **ScaFi** (a Scala-based Aggregate Computing DSL). It uses a local LLM (**Llama 3.2 via Ollama**) paired with real-time Scala compilation feedback (`sbt`) to generate syntactically valid and idiomatic aggregate programming code.

For complete methodology, architectural details, and experimental evaluation, please refer to [`FINAL_REPORT.md`](FINAL_REPORT.md).
